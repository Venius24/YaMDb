# YaMDb — База отзывов о произведениях

**YaMDb** — это REST API сервис, который собирает отзывы пользователей на произведения. Сами произведения делятся на категории (Книги, Фильмы, Музыка) и жанры. Пользователи могут оставлять к произведениям текстовые отзывы и ставить оценку в диапазоне от 1 до 10.

## Ключевые возможности

* **Система отзывов**: Пользователи могут оценивать произведения и комментировать отзывы других людей.
* **Рейтинги**: На основе оценок автоматически вычисляется средний рейтинг произведения.
* **JWT-авторизация**: Безопасная работа через токены.
* **Разграничение прав**: Система ролей (Пользователь, Модератор, Админ) для управления контентом.
* **Фильтрация и поиск**: Удобный поиск произведений по жанрам, категориям, году выпуска и названиям.

## Стек технологий

* **Python 3.9**
* **Django 3.2**
* **Django REST Framework 3.12**
* **SimpleJWT** (Авторизация по токенам)
* **Django Filter** (Гибкая фильтрация)
* **DRF Nested Routers** (Вложенная маршрутизация для отзывов и комментариев)

## Как запустить проект

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/your_username/api_yamdb.git
cd api_yamdb

```


2. **Cоздайте и активируйте виртуальное окружение:**
```bash
python3 -m venv venv
source venv/bin/activate  # Для Linux/macOS
# venv\Scripts\activate   # Для Windows

```


3. **Установите зависимости:**
```bash
pip install -r requirements.txt

```


4. **Выполните миграции:**
```bash
python3 manage.py migrate

```


5. **Запустите сервер:**
```bash
python3 manage.py runserver

```

## Алгоритм регистрации пользователей

1. Пользователь отправляет запрос с параметрами `email` и `username` на эндпоинт `/api/v1/auth/signup/`.
2. Сервис отправляет письмо с кодом подтверждения (`confirmation_code`) на указанный адрес.
3. Пользователь отправляет запрос с `username` и `confirmation_code` на эндпоинт `/api/v1/auth/token/`, в ответе получает JWT-токен.

---

## Примеры эндпоинтов

| URL | Метод | Описание |
| --- | --- | --- |
| `/api/v1/auth/signup/` | POST | Регистрация нового пользователя |
| `/api/v1/titles/` | GET | Получить список всех произведений |
| `/api/v1/titles/{title_id}/reviews/` | GET/POST | Получить отзывы или оставить свой |
| `/api/v1/categories/` | GET/POST | Список категорий (создание доступно только админу) |

## Роли пользователей

* **Аноним** — может просматривать описания произведений, отзывы и комментарии.
* **Аутентифицированный пользователь** — может читать всё, публиковать отзывы и ставить оценки, комментировать чужие отзывы, редактировать свои отзывы и комментарии.
* **Модератор** — те же права, что и у пользователя, плюс право удалять и редактировать любые отзывы и комментарии.
* **Администратор** — полные права на управление всем контентом и пользователями. Может назначать роли.
